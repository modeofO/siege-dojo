# 1v1 Mode Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a 1v1 game mode with dedicated Cairo contracts and a terminal CLI that plays on Sepolia via Cartridge Controller sessions.

**Architecture:** Three new Cairo systems (`actions_1v1`, `commit_reveal_1v1`, `resolution_1v1`) with two new models (`MatchState1v1`, `RoundMoves1v1`), reusing existing `Commitment`, `NodeState`, and `MatchCounter`. A standalone TypeScript CLI in `scripts/siege-cli/` handles interactive and JSON-mode play with Cartridge session auth (default) and raw private key fallback.

**Tech Stack:** Cairo 2.13.1 / Dojo v1.8.0, TypeScript / starknet.js v8, `@cartridge/controller` SessionProvider

**Spec:** `docs/superpowers/specs/2026-04-02-1v1-mode-design.md`

```cairo
// src/systems/actions_1v1.cairo
use starknet::ContractAddress;

#[starknet::interface]
pub trait IActions1v1<T> {
    fn create_match_1v1(
        ref self: T,
        player_a: ContractAddress,
        player_b: ContractAddress,
    ) -> u64;
    fn get_budget_1v1(self: @T, match_id: u64, is_player_a: bool) -> u8;
}

#[dojo::contract]
pub mod actions_1v1 {
    use starknet::ContractAddress;
    use dojo::model::ModelStorage;
    use dojo::event::EventStorage;
    use siege_dojo::models::match_state::MatchStatus;
    use siege_dojo::models::match_state_1v1::MatchState1v1;
    use siege_dojo::models::node_state::{NodeState, NodeOwner};
    use siege_dojo::models::match_counter::MatchCounter;
    use siege_dojo::models::events::MatchCreated1v1;

    #[generate_trait]
    impl InternalImpl of InternalTrait {
        fn world_default(self: @ContractState) -> dojo::world::WorldStorage {
            self.world(@"siege_dojo")
        }
    }

    #[abi(embed_v0)]
    impl Actions1v1Impl of super::IActions1v1<ContractState> {
        fn create_match_1v1(
            ref self: ContractState,
            player_a: ContractAddress,
            player_b: ContractAddress,
        ) -> u64 {
            let mut world = self.world_default();
            let mut counter: MatchCounter = world.read_model(0_u8);
            let match_id = counter.count + 1;
            counter.count = match_id;
            world.write_model(@counter);

            world.write_model(@MatchState1v1 {
                match_id,
                player_a,
                player_b,
                vault_a_hp: 50,
                vault_b_hp: 50,
                current_round: 1,
                status: MatchStatus::Active,
            });

            let mut i: u8 = 0;
            while i < 3 {
                world.write_model(@NodeState {
                    match_id,
                    node_index: i,
                    owner: NodeOwner::None,
                });
                i += 1;
            };

            world.emit_event(@MatchCreated1v1 {
                match_id,
                player_a,
                player_b,
            });

            match_id
        }

        fn get_budget_1v1(self: @ContractState, match_id: u64, is_player_a: bool) -> u8 {
            let world = self.world_default();
            let target = if is_player_a { NodeOwner::TeamA } else { NodeOwner::TeamB };
            let mut bonus: u8 = 0;
            let mut i: u8 = 0;
            while i < 3 {
                let node: NodeState = world.read_model((match_id, i));
                if node.owner == target {
                    bonus += 1;
                }
                i += 1;
            };
            10 + bonus
        }
    }
}
```

```cairo
// src/systems/resolution_1v1.cairo
#[starknet::interface]
pub trait IResolution1v1<T> {
    fn resolve_round(ref self: T, match_id: u64);
}

#[dojo::contract]
pub mod resolution_1v1 {
    use dojo::model::ModelStorage;
    use dojo::event::EventStorage;
    use siege_dojo::models::match_state::MatchStatus;
    use siege_dojo::models::match_state_1v1::MatchState1v1;
    use siege_dojo::models::node_state::{NodeState, NodeOwner};
    use siege_dojo::models::round_moves_1v1::RoundMoves1v1;
    use siege_dojo::models::events::{RoundResolved, MatchFinished};

    #[generate_trait]
    impl InternalImpl of InternalTrait {
        fn world_default(self: @ContractState) -> dojo::world::WorldStorage {
            self.world(@"siege_dojo")
        }
    }

    #[abi(embed_v0)]
    impl Resolution1v1Impl of super::IResolution1v1<ContractState> {
        fn resolve_round(ref self: ContractState, match_id: u64) {
            let mut world = self.world_default();
            let mut state: MatchState1v1 = world.read_model(match_id);
            assert(state.status == MatchStatus::Active, 'Match not active');

            let round = state.current_round;
            let rm: RoundMoves1v1 = world.read_model((match_id, round));
            assert(rm.reveal_count == 2, 'Not all revealed');

            // Damage to vault B (Player A attacks, Player B defends)
            let mut damage_to_b: u8 = 0;
            if rm.a_p0 > rm.b_g0 {
                damage_to_b += rm.a_p0 - rm.b_g0;
            }
            if rm.a_p1 > rm.b_g1 {
                damage_to_b += rm.a_p1 - rm.b_g1;
            }
            if rm.a_p2 > rm.b_g2 {
                damage_to_b += rm.a_p2 - rm.b_g2;
            }

            // Damage to vault A (Player B attacks, Player A defends)
            let mut damage_to_a: u8 = 0;
            if rm.b_p0 > rm.a_g0 {
                damage_to_a += rm.b_p0 - rm.a_g0;
            }
            if rm.b_p1 > rm.a_g1 {
                damage_to_a += rm.b_p1 - rm.a_g1;
            }
            if rm.b_p2 > rm.a_g2 {
                damage_to_a += rm.b_p2 - rm.a_g2;
            }

            // Repairs (capped at 3)
            let repair_a = if rm.a_repair > 3 { 3_u8 } else { rm.a_repair };
            let repair_b = if rm.b_repair > 3 { 3_u8 } else { rm.b_repair };

            let mut hp_a = state.vault_a_hp;
            let mut hp_b = state.vault_b_hp;

            // Repair first
            if hp_a + repair_a > 50 { hp_a = 50; } else { hp_a = hp_a + repair_a; }
            if hp_b + repair_b > 50 { hp_b = 50; } else { hp_b = hp_b + repair_b; }

            // Then damage
            if damage_to_a >= hp_a { hp_a = 0; } else { hp_a = hp_a - damage_to_a; }
            if damage_to_b >= hp_b { hp_b = 0; } else { hp_b = hp_b - damage_to_b; }

            state.vault_a_hp = hp_a;
            state.vault_b_hp = hp_b;

            // Node contests
            let mut n: u8 = 0;
            while n < 3 {
                let (contest_a, contest_b) = if n == 0 {
                    (rm.a_nc0, rm.b_nc0)
                } else if n == 1 {
                    (rm.a_nc1, rm.b_nc1)
                } else {
                    (rm.a_nc2, rm.b_nc2)
                };

                if contest_a > contest_b {
                    world.write_model(@NodeState { match_id, node_index: n, owner: NodeOwner::TeamA });
                } else if contest_b > contest_a {
                    world.write_model(@NodeState { match_id, node_index: n, owner: NodeOwner::TeamB });
                }
                n += 1;
            };

            world.emit_event(@RoundResolved {
                match_id,
                round,
                vault_a_hp: hp_a.into(),
                vault_b_hp: hp_b.into(),
            });

            // Win condition
            if hp_a == 0 || hp_b == 0 {
                state.status = MatchStatus::Finished;
                let winner_team: u8 = if hp_b == 0 && hp_a > 0 {
                    1_u8 // Player A wins
                } else if hp_a == 0 && hp_b > 0 {
                    2_u8 // Player B wins
                } else {
                    0_u8 // Draw
                };
                world.emit_event(@MatchFinished { match_id, winner_team });
            } else if state.current_round >= 10 {
                state.status = MatchStatus::Finished;
                let winner_team: u8 = if hp_a > hp_b {
                    1_u8
                } else if hp_b > hp_a {
                    2_u8
                } else {
                    0_u8
                };
                world.emit_event(@MatchFinished { match_id, winner_team });
            } else {
                state.current_round = round + 1;
            }

            world.write_model(@state);
        }
    }
}
```

```cairo
// src/systems/commit_reveal_1v1.cairo
#[starknet::interface]
pub trait ICommitReveal1v1<T> {
    fn commit(ref self: T, match_id: u64, commitment: felt252);
    fn reveal(
        ref self: T,
        match_id: u64,
        salt: felt252,
        p0: u8, p1: u8, p2: u8,
        g0: u8, g1: u8, g2: u8,
        repair: u8,
        nc0: u8, nc1: u8, nc2: u8,
    );
    fn force_timeout(ref self: T, match_id: u64);
}

#[dojo::contract]
pub mod commit_reveal_1v1 {
    use core::poseidon::PoseidonTrait;
    use core::hash::HashStateTrait;
    use starknet::{ContractAddress, get_caller_address, get_block_timestamp};
    use dojo::model::ModelStorage;
    use dojo::world::WorldStorageTrait;
    use dojo::event::EventStorage;
    use siege_dojo::models::match_state::MatchStatus;
    use siege_dojo::models::match_state_1v1::MatchState1v1;
    use siege_dojo::models::node_state::{NodeState, NodeOwner};
    use siege_dojo::models::commitment::Commitment;
    use siege_dojo::models::round_moves_1v1::RoundMoves1v1;
    use siege_dojo::systems::resolution_1v1::{IResolution1v1Dispatcher, IResolution1v1DispatcherTrait};
    use siege_dojo::models::events::{MoveCommitted, MoveRevealed};

    const COMMIT_TIMEOUT: u64 = 300;
    const REVEAL_TIMEOUT: u64 = 300;

    const ROLE_A: u8 = 0;
    const ROLE_B: u8 = 1;

    #[generate_trait]
    impl InternalImpl of InternalTrait {
        fn world_default(self: @ContractState) -> dojo::world::WorldStorage {
            self.world(@"siege_dojo")
        }
    }

    fn get_player_role(state: @MatchState1v1, caller: ContractAddress) -> u8 {
        if caller == *state.player_a {
            ROLE_A
        } else if caller == *state.player_b {
            ROLE_B
        } else {
            panic!("Not a player in this match")
        }
    }

    fn calc_budget(world: @dojo::world::WorldStorage, match_id: u64, is_player_a: bool) -> u8 {
        let target = if is_player_a { NodeOwner::TeamA } else { NodeOwner::TeamB };
        let mut bonus: u8 = 0;
        let mut i: u8 = 0;
        while i < 3 {
            let node: NodeState = world.read_model((match_id, i));
            if node.owner == target {
                bonus += 1;
            }
            i += 1;
        };
        10 + bonus
    }

    #[abi(embed_v0)]
    impl CommitReveal1v1Impl of super::ICommitReveal1v1<ContractState> {
        fn commit(ref self: ContractState, match_id: u64, commitment: felt252) {
            let mut world = self.world_default();
            let caller = get_caller_address();
            let state: MatchState1v1 = world.read_model(match_id);
            assert(state.status == MatchStatus::Active, 'Match not active');

            let role = get_player_role(@state, caller);
            let round = state.current_round;

            let existing: Commitment = world.read_model((match_id, round, role));
            assert(!existing.committed, 'Already committed');

            world.write_model(@Commitment {
                match_id, round, role,
                hash: commitment,
                committed: true,
                revealed: false,
            });

            let mut rm: RoundMoves1v1 = world.read_model((match_id, round));
            rm.commit_count += 1;

            if rm.commit_count == 1 {
                rm.commit_deadline = get_block_timestamp() + COMMIT_TIMEOUT;
            }
            if rm.commit_count == 2 {
                rm.reveal_deadline = get_block_timestamp() + REVEAL_TIMEOUT;
            }

            world.write_model(@rm);

            world.emit_event(@MoveCommitted { match_id, round, role });
        }

        fn reveal(
            ref self: ContractState,
            match_id: u64,
            salt: felt252,
            p0: u8, p1: u8, p2: u8,
            g0: u8, g1: u8, g2: u8,
            repair: u8,
            nc0: u8, nc1: u8, nc2: u8,
        ) {
            let mut world = self.world_default();
            let caller = get_caller_address();
            let state: MatchState1v1 = world.read_model(match_id);
            assert(state.status == MatchStatus::Active, 'Match not active');

            let role = get_player_role(@state, caller);
            let round = state.current_round;

            let rm: RoundMoves1v1 = world.read_model((match_id, round));
            assert(rm.commit_count == 2, 'Not all committed');

            let mut c: Commitment = world.read_model((match_id, round, role));
            assert(c.committed, 'Not committed');
            assert(!c.revealed, 'Already revealed');

            // Verify hash
            let mut h = PoseidonTrait::new();
            h = h.update(salt);
            h = h.update(p0.into());
            h = h.update(p1.into());
            h = h.update(p2.into());
            h = h.update(g0.into());
            h = h.update(g1.into());
            h = h.update(g2.into());
            h = h.update(repair.into());
            h = h.update(nc0.into());
            h = h.update(nc1.into());
            h = h.update(nc2.into());
            let computed = h.finalize();
            assert(computed == c.hash, 'Invalid reveal');

            // Budget check
            let is_player_a = role == ROLE_A;
            let budget = calc_budget(@world, match_id, is_player_a);
            let total: u16 = p0.into() + p1.into() + p2.into()
                + g0.into() + g1.into() + g2.into()
                + repair.into()
                + nc0.into() + nc1.into() + nc2.into();
            assert(total <= budget.into(), 'Over budget');

            c.revealed = true;
            world.write_model(@c);

            let mut rm: RoundMoves1v1 = world.read_model((match_id, round));
            rm.reveal_count += 1;

            if role == ROLE_A {
                rm.a_p0 = p0; rm.a_p1 = p1; rm.a_p2 = p2;
                rm.a_g0 = g0; rm.a_g1 = g1; rm.a_g2 = g2;
                rm.a_repair = repair;
                rm.a_nc0 = nc0; rm.a_nc1 = nc1; rm.a_nc2 = nc2;
            } else {
                rm.b_p0 = p0; rm.b_p1 = p1; rm.b_p2 = p2;
                rm.b_g0 = g0; rm.b_g1 = g1; rm.b_g2 = g2;
                rm.b_repair = repair;
                rm.b_nc0 = nc0; rm.b_nc1 = nc1; rm.b_nc2 = nc2;
            }

            world.write_model(@rm);

            world.emit_event(@MoveRevealed { match_id, round, role });

            if rm.reveal_count == 2 {
                let (res_addr, _) = world.dns(@"resolution_1v1").unwrap();
                let res = IResolution1v1Dispatcher { contract_address: res_addr };
                res.resolve_round(match_id);
            }
        }

        fn force_timeout(ref self: ContractState, match_id: u64) {
            let mut world = self.world_default();
            let state: MatchState1v1 = world.read_model(match_id);
            assert(state.status == MatchStatus::Active, 'Match not active');

            let round = state.current_round;
            let now = get_block_timestamp();
            let mut rm: RoundMoves1v1 = world.read_model((match_id, round));

            if rm.commit_count < 2 && rm.commit_count > 0 {
                assert(now >= rm.commit_deadline, 'Commit deadline not reached');
                let mut r: u8 = 0;
                while r < 2 {
                    let c: Commitment = world.read_model((match_id, round, r));
                    if !c.committed {
                        world.write_model(@Commitment {
                            match_id, round, role: r,
                            hash: 0,
                            committed: true,
                            revealed: false,
                        });
                    }
                    r += 1;
                };
                rm.commit_count = 2;
                rm.reveal_deadline = now + REVEAL_TIMEOUT;
                world.write_model(@rm);
            }

            if rm.commit_count == 2 && rm.reveal_count < 2 {
                assert(now >= rm.reveal_deadline, 'Reveal deadline not reached');
                let mut r: u8 = 0;
                while r < 2 {
                    let c: Commitment = world.read_model((match_id, round, r));
                    if !c.revealed {
                        world.write_model(@Commitment {
                            match_id, round, role: r,
                            hash: c.hash,
                            committed: true,
                            revealed: true,
                        });
                        rm.reveal_count += 1;
                    }
                    r += 1;
                };
                world.write_model(@rm);

                let (res_addr, _) = world.dns(@"resolution_1v1").unwrap();
                let res = IResolution1v1Dispatcher { contract_address: res_addr };
                res.resolve_round(match_id);
            }
        }
    }
}
```

```typescript
// scripts/siege-cli/auth.ts
import { Account, RpcProvider, type AccountInterface } from "starknet";
import { CONTRACTS, RPC_URL, CHAIN_ID } from "./chain.ts";

export interface AuthConfig {
  usePrivateKey: boolean;
}

export async function getAccount(config: AuthConfig): Promise<AccountInterface> {
  if (config.usePrivateKey) {
    return getPrivateKeyAccount();
  }
  return getCartridgeAccount();
}

function getPrivateKeyAccount(): AccountInterface {
  const address = process.env.DOJO_ACCOUNT_ADDRESS;
  const pk = process.env.DOJO_PRIVATE_KEY;
  if (!address || !pk) {
    console.error("Error: DOJO_ACCOUNT_ADDRESS and DOJO_PRIVATE_KEY must be set with --use-private-key");
    process.exit(1);
  }
  const provider = new RpcProvider({ nodeUrl: RPC_URL });
  return new Account({ provider, address, signer: pk });
}

async function getCartridgeAccount(): Promise<AccountInterface> {
  // Dynamic import to avoid requiring @cartridge/controller when using private key mode
  const { default: SessionProvider } = await import("@cartridge/controller/session/node");
  const path = await import("path");

  const storagePath = process.env.CARTRIDGE_STORAGE_PATH || path.join(process.cwd(), ".cartridge");

  const provider = new SessionProvider({
    rpc: RPC_URL,
    chainId: CHAIN_ID,
    policies: {
      contracts: {
        [CONTRACTS.ACTIONS_1V1]: {
          methods: [
            { name: "Create 1v1 Match", entrypoint: "create_match_1v1" },
          ],
        },
        [CONTRACTS.COMMIT_REVEAL_1V1]: {
          methods: [
            { name: "Commit Move", entrypoint: "commit" },
            { name: "Reveal Move", entrypoint: "reveal" },
          ],
        },
      },
    },
    basePath: storagePath,
  });

  const account = await provider.connect();
  if (!account) {
    console.log("Please complete the session creation in your browser, then re-run.");
    process.exit(0);
  }
  return account;
}
```

```typescript
// scripts/siege-cli/game.ts
import * as readline from "readline";
import type { MoveAllocation, MatchInfo } from "./chain.ts";

const rl = readline.createInterface({ input: process.stdin, output: process.stdout });

function ask(prompt: string): Promise<string> {
  return new Promise(resolve => rl.question(prompt, resolve));
}

function parseNumbers(input: string, count: number): number[] | null {
  const parts = input.trim().split(/\s+/).map(Number);
  if (parts.length !== count || parts.some(n => isNaN(n) || n < 0)) return null;
  return parts;
}

export function computeBudget(nodes: [string, string, string], isPlayerA: boolean): number {
  const team = isPlayerA ? "TeamA" : "TeamB";
  const bonus = nodes.filter(n => n === team).length;
  return 10 + bonus;
}

export function validateMove(move: MoveAllocation, budget: number): string | null {
  const total = move.attack.reduce((a, b) => a + b, 0)
    + move.defense.reduce((a, b) => a + b, 0)
    + move.repair
    + move.nodes.reduce((a, b) => a + b, 0);
  if (total > budget) return `Over budget: ${total}/${budget}`;
  if (move.repair > 3) return `Repair cannot exceed 3 (got ${move.repair})`;
  const allVals = [...move.attack, ...move.defense, move.repair, ...move.nodes];
  if (allVals.some(v => v < 0)) return "Values cannot be negative";
  return null;
}

export function parseJsonMove(json: string): MoveAllocation | null {
  try {
    const obj = JSON.parse(json);
    if (!Array.isArray(obj.attack) || obj.attack.length !== 3) return null;
    if (!Array.isArray(obj.defense) || obj.defense.length !== 3) return null;
    if (typeof obj.repair !== "number") return null;
    if (!Array.isArray(obj.nodes) || obj.nodes.length !== 3) return null;
    return {
      attack: obj.attack as [number, number, number],
      defense: obj.defense as [number, number, number],
      repair: obj.repair,
      nodes: obj.nodes as [number, number, number],
    };
  } catch {
    return null;
  }
}

export async function promptMove(budget: number): Promise<MoveAllocation> {
  while (true) {
    console.log(`\n--- Allocate your budget (${budget} points) ---`);

    const atkInput = await ask("Attack (p0 p1 p2): ");
    const atk = parseNumbers(atkInput, 3);
    if (!atk) { console.log("Invalid input. Enter 3 numbers separated by spaces."); continue; }

    const defInput = await ask("Defense (g0 g1 g2): ");
    const def = parseNumbers(defInput, 3);
    if (!def) { console.log("Invalid input. Enter 3 numbers separated by spaces."); continue; }

    const repairInput = await ask("Repair (0-3): ");
    const repair = Number(repairInput);
    if (isNaN(repair) || repair < 0) { console.log("Invalid input. Enter a number 0-3."); continue; }

    const nodesInput = await ask("Nodes (nc0 nc1 nc2): ");
    const nodes = parseNumbers(nodesInput, 3);
    if (!nodes) { console.log("Invalid input. Enter 3 numbers separated by spaces."); continue; }

    const move: MoveAllocation = {
      attack: atk as [number, number, number],
      defense: def as [number, number, number],
      repair,
      nodes: nodes as [number, number, number],
    };

    const error = validateMove(move, budget);
    if (error) { console.log(error); continue; }

    const total = atk.reduce((a, b) => a + b, 0) + def.reduce((a, b) => a + b, 0) + repair + nodes.reduce((a, b) => a + b, 0);
    console.log(`Total: ${total}/${budget}`);
    return move;
  }
}

function nodeLabel(owner: string): string {
  if (owner === "TeamA") return "A";
  if (owner === "TeamB") return "B";
  return "-";
}

export function displayMatchStatus(info: MatchInfo, isPlayerA: boolean) {

  const nodeDisplay = info.nodes.map(n => `[${nodeLabel(n)}]`).join(" ");

  console.log(`Nodes: ${nodeDisplay}`);
  console.log(`You are: ${role}`);
}

export function displayRoundResults(
  myMove: MoveAllocation,
  theirAttack: number[],
  theirDefense: number[],
  newInfo: MatchInfo,
  isPlayerA: boolean,
) {
  const myAtk = myMove.attack;
  const myDef = myMove.defense;

  const damageDealt = myAtk.reduce((sum, a, i) => sum + Math.max(0, a - theirDefense[i]), 0);
  const damageTaken = theirAttack.reduce((sum, a, i) => sum + Math.max(0, a - myDef[i]), 0);

  console.log(`\n--- Round Results ---`);
  console.log(`Your attack [${myAtk}] vs their defense [${theirDefense}]`);
  console.log(`  Damage dealt: ${damageDealt}`);
  console.log(`Their attack [${theirAttack}] vs your defense [${myDef}]`);
  console.log(`  Damage taken: ${damageTaken}`);
  console.log(`Vault A: ${newInfo.vaultAHp} HP | Vault B: ${newInfo.vaultBHp} HP`);

  if (newInfo.status === "Finished") {
    if (newInfo.vaultAHp === 0 && newInfo.vaultBHp === 0) {
      console.log("\nDraw!");
    } else if (newInfo.vaultAHp === 0) {
      console.log(isPlayerA ? "\nYou lost!" : "\nYou won!");
    } else if (newInfo.vaultBHp === 0) {
      console.log(isPlayerA ? "\nYou won!" : "\nYou lost!");
    } else {
      // Round 10 HP comparison
      const winner = newInfo.vaultAHp > newInfo.vaultBHp ? "A" : "B";
      const isWinner = (winner === "A") === isPlayerA;
      console.log(isWinner ? "\nYou won by HP!" : "\nYou lost by HP!");
    }
  }
}
```

```typescript
// scripts/siege-cli/siege-cli.ts
import { RpcProvider } from "starknet";
import { getAccount, type AuthConfig } from "./auth.ts";
import {
  RPC_URL, fetchMatchState, fetchRoundStatus, fetchRoundMoves,
  computeCommitment, generateSalt, createMatch,
  commitMove, revealMove, type MoveAllocation,
} from "./chain.ts";
import {
  computeBudget, validateMove, parseJsonMove,
  promptMove, displayMatchStatus, displayRoundResults, closePrompt,
} from "./game.ts";

// --- Arg parsing ---

function parseArgs() {
  const args = process.argv.slice(2);
  const flags: Record<string, string | boolean> = {};

  for (let i = 0; i < args.length; i++) {
    const arg = args[i];
    if (arg === "--create") flags.create = true;
    else if (arg === "--use-private-key") flags.usePrivateKey = true;
    else if (arg === "--match" && args[i + 1]) flags.match = args[++i];
    else if (arg === "--opponent" && args[i + 1]) flags.opponent = args[++i];
    else if (arg === "--json" && args[i + 1]) flags.json = args[++i];
    else if (arg === "--player" && args[i + 1]) flags.player = args[++i];
  }

  return flags;
}

// --- Polling ---

async function waitFor(
  check: () => Promise<boolean>,
  label: string,
  intervalMs = 3000,
): Promise<void> {
  process.stdout.write(`${label}...`);
  while (true) {
    if (await check()) { console.log(" done."); return; }
    process.stdout.write(".");
    await new Promise(r => setTimeout(r, intervalMs));
  }
}

// --- Main ---

async function main() {
  const flags = parseArgs();
  const authConfig: AuthConfig = { usePrivateKey: !!flags.usePrivateKey };

  console.log("=== Siege Dojo 1v1 ===\n");

  // Authenticate
  const account = await getAccount(authConfig);
  const myAddress = account.address;
  console.log(`Wallet: ${myAddress.slice(0, 10)}...${myAddress.slice(-6)}`);

  const provider = new RpcProvider({ nodeUrl: RPC_URL });

  let matchId: number;

  // Create or join
  if (flags.create) {
    if (!flags.opponent) {
      console.error("Error: --create requires --opponent <address>");
      process.exit(1);
    }
    console.log("Creating match...");
    const txHash = await createMatch(account, flags.opponent as string);
    await provider.waitForTransaction(txHash);

    // Wait for Torii to index, then find the match
    await new Promise(r => setTimeout(r, 3000));

    // Find the latest match by polling
    // We need to find the match ID from the transaction events
    // For simplicity, poll Torii for the most recent match involving our address
    console.log("Waiting for Torii to index...");
    let info: Awaited<ReturnType<typeof fetchMatchState>> = null;
    for (let attempt = 0; attempt < 10; attempt++) {
      // Try recent match IDs (count backwards from a high number)
      // Better approach: query all matches and find ours
      await new Promise(r => setTimeout(r, 2000));
      // For now, we'll need the match ID from the counter
      // This is a known limitation — will improve after initial testing
      break;
    }

    console.log(`Match created! Transaction: ${txHash}`);
    console.log("Note: Check Torii to find your match ID, then rejoin with --match <id>");
    console.log("(Auto-detection of match ID from transaction will be added in a future update)");
    process.exit(0);

  } else if (flags.match) {
    matchId = Number(flags.match);
  } else {
    console.error("Error: specify --create --opponent <addr> or --match <id>");
    process.exit(1);
  }

  // Fetch match state
  let info = await fetchMatchState(matchId!);
  if (!info) {
    console.error(`Match ${matchId!} not found`);
    process.exit(1);
  }

  // Determine role
  const normalizeAddr = (a: string) => "0x" + BigInt(a).toString(16);
  const myNorm = normalizeAddr(myAddress);
  const isPlayerA = normalizeAddr(info.playerA) === myNorm;
  const isPlayerB = normalizeAddr(info.playerB) === myNorm;
  if (!isPlayerA && !isPlayerB) {
    console.error("Your address doesn't match either player in this match");
    process.exit(1);
  }

  console.log(`\n=== Match #${matchId!} ===`);

  // JSON mode: single round then exit
  if (flags.json) {
    const move = parseJsonMove(flags.json as string);
    if (!move) {
      console.error("Invalid JSON move. Expected: {\"attack\":[n,n,n],\"defense\":[n,n,n],\"repair\":n,\"nodes\":[n,n,n]}");
      process.exit(1);
    }
    const budget = computeBudget(info.nodes, isPlayerA);
    const err = validateMove(move, budget);
    if (err) { console.error(err); process.exit(1); }

    await playRound(account, provider, matchId!, info, isPlayerA, move);
    process.exit(0);
  }

  // Interactive mode: loop rounds
  while (info && info.status === "Active") {
    displayMatchStatus(info, isPlayerA);
    const budget = computeBudget(info.nodes, isPlayerA);
    const move = await promptMove(budget);

    await playRound(account, provider, matchId!, info, isPlayerA, move);

    // Refresh state
    info = await fetchMatchState(matchId!);
    if (info?.status === "Finished") {
      break;
    }
  }

  console.log("\nMatch complete!");
  closePrompt();
}

async function playRound(
  account: Awaited<ReturnType<typeof getAccount>>,
  provider: RpcProvider,
  matchId: number,
  info: NonNullable<Awaited<ReturnType<typeof fetchMatchState>>>,
  isPlayerA: boolean,
  move: MoveAllocation,
) {
  const round = info.currentRound;
  const salt = generateSalt();

  // Commit
  const commitment = computeCommitment(salt, move);
  process.stdout.write("Committing...");
  const commitTx = await commitMove(account, matchId, commitment);
  await provider.waitForTransaction(commitTx);
  console.log(" done.");

  // Wait for opponent commit
  await waitFor(
    async () => (await fetchRoundStatus(matchId, round)).commitCount >= 2,
    "Waiting for opponent to commit",
  );

  // Reveal
  process.stdout.write("Revealing...");
  const revealTx = await revealMove(account, matchId, salt, move);
  await provider.waitForTransaction(revealTx);
  console.log(" done.");

  // Wait for opponent reveal (triggers resolution)
  await waitFor(
    async () => (await fetchRoundStatus(matchId, round)).revealCount >= 2,
    "Waiting for opponent to reveal",
  );

  // Wait for Torii to index resolution
  await new Promise(r => setTimeout(r, 3000));

  // Show results
  const moves = await fetchRoundMoves(matchId, round);
  const newInfo = await fetchMatchState(matchId);
  if (moves && newInfo) {
    const theirAttack = isPlayerA ? moves.bAttack : moves.aAttack;
    const theirDefense = isPlayerA ? moves.bDefense : moves.aDefense;
    displayRoundResults(move, theirAttack, theirDefense, newInfo, isPlayerA);
  }
}

main().catch(err => {
  console.error("Error:", err.message || err);
  process.exit(1);
});
```

## 1v1 Mode

Simplified 1v1 game mode for mechanics testing. Each player controls both attack and defense with a shared budget of 10 (+node bonuses). Vault HP starts at 50.

Same world as 2v2 (`siege_dojo`). Systems: `actions_1v1`, `commit_reveal_1v1`, `resolution_1v1`. Models: `MatchState1v1`, `RoundMoves1v1`. Reuses `Commitment`, `NodeState`, `MatchCounter`.

Each player splits their budget across:
- Attack: 3 pressure points (p0, p1, p2)
- Defense: 3 gates (g0, g1, g2) + repair (max 3)
- Nodes: 3 node contests (nc0, nc1, nc2)
